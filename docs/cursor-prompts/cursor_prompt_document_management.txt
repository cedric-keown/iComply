CURSOR PROMPT: DOCUMENT MANAGEMENT & 5-YEAR RETENTION MODULE
============================================================

Create a fully functional, realistic HTML mockup for the Document Management module of a South African FAIS broker compliance portal. This module manages all compliance documents with mandatory 5-year retention as per FAIS Act requirements, version control, secure storage, and comprehensive audit trails.

- Fully responsive design (mobile-first)
