CURSOR PROMPT: FICA CLIENT VERIFICATION WORKFLOW MODULE
=======================================================

Create a fully functional, realistic HTML mockup for the FICA (Financial Intelligence Centre Act) Client Verification Workflow module of a South African broker compliance portal. This module handles KYC (Know Your Customer), risk assessment, PEP/sanctions screening, and ongoing monitoring as per FICA Act 38 of 2001 requirements.

- Primary (Teal/Turquoise): #5CBDB4

- Background: #FFFFFF

- Warning: #FFC107

- Info: #17A2B8

**Framework & Theme:**
- Use Bootstrap 5+ with Phoenix v1.23.0 theme: https://prium.github.io/phoenix/v1.23.0/index.html
- Fully responsive design (mobile-first)
- Vanilla JavaScript only (no frameworks)
- South African locale (ID format, phone +27, currency ZAR, dates DD/MM/YYYY)
