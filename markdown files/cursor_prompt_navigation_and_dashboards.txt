CURSOR PROMPT: NAVIGATION SYSTEM & LANDING PAGE DASHBOARDS
===========================================================

Create a fully functional, realistic HTML mockup for the complete Navigation System and Role-Based Landing Page Dashboards for a South African FAIS broker compliance portal. This includes the main navigation structure, sidebar menus, top bar, and customized dashboards for each of the 5 user roles.

## BRAND & DESIGN REQUIREMENTS

**Brand Colors:**
- Primary (Teal/Turquoise): #5CBDB4
- Text (Charcoal Grey): #4A4A4A
- Background: #FFFFFF
- Light Grey: #F8F9FA
- Success: #28A745
- Warning: #FFC107
- Danger: #DC3545
- Info: #17A2B8

**Framework & Theme:**
- Use Bootstrap 5+ with Phoenix v1.23.0 theme: https://prium.github.io/phoenix/v1.23.0/index.html
- Fully responsive design (mobile-first)
- Vanilla JavaScript only (no frameworks)
- South African locale (ZAR currency, DD/MM/YYYY dates)

### BREADCRUMB NAVIGATION

**Location:** Below top navbar, above main content

**Structure:**
```html
<nav aria-label="breadcrumb">
  <ol class="breadcrumb">
    <li><a href="#"><i class="bi bi-house"></i> Home</a></li>
    <li><a href="#">Representatives</a></li>
    <li class="active">Thabo Mokoena</li>
  </ol>
</nav>
```

**Features:**
- Clickable path segments
- Home icon at start
- Auto-generate from route
- Max depth: 4 levels
- Truncate on mobile
- Show current page in bold

- Home > Dashboard

## PART 2: ROLE-BASED LANDING PAGE DASHBOARDS

### Common Dashboard Components

**Dashboard Grid System:**
- Responsive grid layout
- 12-column Bootstrap grid
- Drag & drop to rearrange (future enhancement)
- Widgets can be minimized/maximized
- Print-friendly view

**Common Widget Types:**

1. **Stat Cards** - Key metrics with icons

### Responsive Behavior

```javascript
// Responsive navigation
const breakpoints = {
    mobile: 576,
    tablet: 768,
    desktop: 992,
    wide: 1200
};

function handleResize() {
    const width = window.innerWidth;
    
    if (width < breakpoints.desktop) {
        // Mobile/tablet
        navigationState.sidebarCollapsed = true;
        showBottomNav();
        hideSidebar();
    } else {
        // Desktop
        hideBottomNav();
        showSidebar();
        // Restore user preference
        navigationState.sidebarCollapsed = localStorage.getItem('sidebarCollapsed') === 'true';
    }
}

window.addEventListener('resize', debounce(handleResize, 250));
```
